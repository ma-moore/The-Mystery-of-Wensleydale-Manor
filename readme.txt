Welcome to The Mystery of Wensleydale Manor

You just woke up in a grave in a swamp. Who are you and where are you? Ask around and find out!

Controls
-Arrow keys to walk around
-Space bar to interact

	Install-win32 - for Windows 32 bit systems
	Install-win64 - for Windows 64 bit systems
	Install-mac - for Macintosh systems
	Install-linux32 - for Linux 32 bit systems
	Install-linux64 - for Linux 64 bit systems
